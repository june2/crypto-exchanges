### 실행방법

```
#npm 모듈 설치
yarn

#test code 실행
$ yarn test

#application 실행
$ yarn start

#CLI 활용

$ yarn start [template파일 위치] [input data 파일 위치] [output 파일 위치]
$ yarn start files/template1.txt files/data.json output.txt

** args 미입력시, files/template1.txt files/data.json output.txt 으로 default 셋업
```
